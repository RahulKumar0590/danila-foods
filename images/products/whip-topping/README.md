# Whip Topping Product Images - Step by Step Guide

## STEP 1: Download Base Images

### Option A: Search and Download
1. Go to Google Images: https://www.google.com/imagesearch
2. Search for: **"whip topping product packaging"** or **"non dairy whip cream package"**
3. Look for images of whipped cream/topping product containers (buckets or pouches)
4. Download 2 different product images:
   - One with white/light packaging (for White Whip)
   - One with cream/beige packaging (for Original Whip)
5. Save them to your Downloads folder

### Option B: Use Stock Photo Sites (Free)
- Unsplash: https://unsplash.com/s/photos/whipped-cream-product
- Pexels: https://pexels.com/search/whipped-cream-container/
- Pixabay: https://pixabay.com/images/search/whipped-cream/

## STEP 2: Edit Images to Add DANILA Branding

### Option A: Using Canva (Easiest - Free Online)
1. Go to https://www.canva.com/
2. Click "Create a design" → "Custom size" → Enter 800 x 800 pixels
3. Upload your downloaded whip topping image
4. Add text elements:
   - **Top of package:** "DANILA" (Font: Bold, Size: 72-80, Color: Red #E74C3C)
   - **Middle:** "White Whip" or "Original Whip" (Font: Bold, Size: 48-56, Color: Dark Blue #2C3E50)
   - **Below that:** "Non Dairy Whip Topping" (Font: Regular, Size: 24-32, Color: Gray)
   - **Bottom:** "1 KG" or "2 KG" in a circle/badge (Font: Bold, Size: 60-72, Color: Red, with white background circle)
5. Download as PNG or JPG
6. Rename file to match required names below

### Option B: Using Microsoft PowerPoint
1. Open PowerPoint → New Blank Presentation
2. Insert → Pictures → Select your downloaded image
3. Insert → Text Box → Add text:
   - Type "DANILA" → Format: Font Size 72, Bold, Red color
   - Type "White Whip" → Font Size 48, Bold, Dark Blue
   - Type "Non Dairy Whip Topping" → Font Size 28, Italic, Gray
   - Type "1 KG" → Font Size 60, Bold, Red, Insert Shape (Circle) behind it
4. Right-click slide → Save as Picture → Save as JPG/PNG
5. Rename the file

### Option C: Using Paint.NET (Free Download)
1. Download from: https://www.getpaint.net/
2. Open your downloaded image
3. Use Text tool (T key) to add:
   - "DANILA" at top (Font: Arial Black, Size: 72, Color: Red)
   - "White Whip" or "Original Whip" in middle (Size: 48, Bold)
   - "Non Dairy Whip Topping" (Size: 24, Italic)
   - "1 KG" or "2 KG" at bottom (Size: 60, Bold, Red)
4. Save as PNG/JPG

### Option D: Using Photoshop/GIMP (Advanced)
1. Open image in Photoshop or GIMP
2. Use Text Tool to add layers with branding
3. Apply effects like drop shadow for better visibility
4. Export as JPG/PNG
